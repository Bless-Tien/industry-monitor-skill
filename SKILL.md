# Industry Monitor - 通用行业监控技能

## 概述

本技能是一个通用的行业监控系统，适用于任何行业的资讯监控和竞争分析。用户可以根据自己的行业特点，自定义监控对象、竞品列表和关键词，系统将自动执行全方位监控并生成专业分析报告。

## 核心功能

### 1. 全面监控覆盖

#### 可自定义监控对象
- **本公司**：您的公司名称、品牌、英文名
- **主要竞品**：行业内主流竞争对手
- **潜在竞品**：新兴品牌和潜在威胁
- **跨界入场者**：可能进入该行业的大公司

#### 标准监控维度（14个）
1. 本公司动态（Critical）
2. 主要竞品动态（Critical）
3. 新兴竞品动态（High）
4. 跨界入场监控（Critical）
5. 行业新闻（High）
6. 技术前沿（High）
7. 学术论文（High）
8. 新品发布（High）
9. 政策法规（Medium）
10. 市场数据（Medium）
11. 融资并购（High）
12. 渠道合作（Medium）
13. 用户反馈（Medium）
14. 负面舆情（Critical）

### 2. 严格时效控制（48小时窗口）

#### 执行前必须确认日期
**第一步：获取当前日期**
```python
# 从系统环境变量获取当前日期
current_date = "YYYY-MM-DD"
target_dates = ["今天", "昨天"]  # 48小时窗口
```

#### 48小时时效保证
- **记录规则**：只记录当天和前一天的信息
- **分析规则**：可以综合分析更早的信息作为背景
- **搜索词要求**：所有搜索词必须包含时间关键词
- **结果过滤**：严格过滤发布时间，只保留48小时内

#### 支持的日期格式
- 标准格式：YYYY-MM-DD、YYYY/MM/DD
- 中文格式：X年X月X日、X月X日、今天、昨天
- 英文格式：Mon DD, YYYY、today、yesterday
- 相对时间：X小时前、X天前

### 3. 智能预警系统

#### 5类预警规则
1. **公司提及预警**：本公司被提及时高亮标注
2. **竞品重大事件**：融资、并购、新品、专利、获奖自动高亮
3. **负面新闻预警**：召回、质量问题、投诉、处罚触发预警
4. **跨界入场预警**：大公司进军、布局、入局该行业
5. **技术突破预警**：突破、首创、专利、创新

### 4. 结构化报告

#### 报告结构（按优先级排序）
1. 📊 执行摘要
2. ⚠️ 重要预警
3. 🏢 本公司相关
4. 🎯 主要竞品动态
5. 🆕 新兴竞品动态
6. 🚀 跨界入场动态
7. 📰 行业新闻
8. 🔬 技术前沿与学术研究
9. 🆕 新品发布
10. 📜 政策法规
11. 📊 市场数据
12. 💰 融资并购
13. 🤝 渠道合作
14. 💬 用户反馈
15. ⚠️ 负面舆情
16. 📈 综合分析
17. 🎯 战略建议
18. ⚠️ 风险预警

## 使用方法

### 首次使用：配置监控对象

**用户说**：
```
我想监控[行业名称]行业
```

**AI会询问**：
1. 您的公司名称和品牌
2. 主要竞品有哪些（3-10家）
3. 需要关注的新兴品牌
4. 可能跨界进入的大公司
5. 特别关注的关键词

**AI会创建**：
- 自定义配置文件
- 专属监控关键词库
- 个性化预警规则

### 标准触发词

**日常触发**：
- "执行监控"
- "查询行业动态"
- "生成监控报告"
- "今天有什么新闻"
- "行业最新资讯"

**特定查询**：
- "查询本公司动态"
- "查询竞品动态"
- "有没有新玩家入场"
- "最新的融资并购"
- "技术前沿有什么突破"

### 执行流程

1. **确认日期**：获取当前系统日期，计算48小时窗口
2. **读取配置**：加载用户自定义的监控配置
3. **执行搜索**：对14个维度执行真实网络搜索（20-30次）
4. **智能过滤**：按时效性（48小时）、相关性过滤
5. **生成报告**：创建结构化Markdown报告
6. **保存报告**：保存到指定目录
7. **返回结果**：展示关键发现和报告位置

## 技术说明

### 搜索策略
- 使用WebSearch工具执行真实搜索
- 每个维度3-5个搜索词
- 总计20-30次搜索请求
- 获取80-120条真实资讯

### 过滤算法
- 时间过滤：严格48小时窗口
- 相关性过滤：关键词匹配度
- 重要性评分：根据来源和内容
- 去重处理：相同新闻合并

### 分析方法
- 关键词提取：TF-IDF算法
- 情感分析：正面/负面/中性
- 趋势判断：时间序列分析
- 关联分析：事件关联挖掘

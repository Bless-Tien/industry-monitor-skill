> 一个强大的AI驱动的行业监控系统，适用于任何行业的竞争情报收集和分析

## 🌟 特性

- ✅ **通用性强**：适用于任何行业
- ✅ **全面监控**：14个维度全覆盖
- ✅ **实时追踪**：48小时时效控制
- ✅ **智能预警**：5类预警规则
- ✅ **自动化**：一键执行，自动生成报告
- ✅ **可定制**：支持自定义监控对象和关键词

## 📋 功能概览

### 监控维度

1. **本公司动态** - 公司新闻、公告、媒体报道
2. **主要竞品** - 竞争对手最新动作
3. **新兴竞品** - 新玩家和潜在威胁
4. **跨界入场** - 大公司进入该行业
5. **行业新闻** - 行业整体动态和趋势
6. **技术前沿** - 技术创新和突破
7. **学术论文** - 最新研究成果
8. **新品发布** - 新产品上市信息
9. **政策法规** - 行业标准和政策
10. **市场数据** - 市场规模和销售数据
11. **融资并购** - 投资和收购动态
12. **渠道合作** - 战略合作和渠道拓展
13. **用户反馈** - 产品评测和用户评价
14. **负面舆情** - 负面新闻和风险预警

### 智能预警

- 🏢 **公司提及预警** - 本公司被提及时高亮
- 🎯 **竞品重大事件** - 融资、并购、新品等自动标注
- ⚠️ **负面新闻预警** - 召回、质量问题等触发警报
- 🚀 **跨界入场预警** - 大公司进军该行业
- 💡 **技术突破预警** - 技术创新和专利

## 🚀 快速开始

### 安装

1. 下载本技能文件
2. 将整个文件夹放到技能目录：
   ```
   ~/.stepfun/skills/industry-monitor/
   ```

### 首次配置

**对AI说**：
```
我想监控[您的行业]行业
```

**AI会询问**：
- 您的公司名称和品牌
- 主要竞品有哪些
- 需要关注的新兴品牌
- 可能跨界进入的大公司

**AI会自动创建**：
- 个性化配置文件
- 专属监控关键词
- 定制预警规则

### 开始使用

**对AI说任一句**：
- "执行监控"
- "查询行业动态"
- "生成监控报告"
- "今天有什么新闻"

**AI会自动**：
1. 执行20-30次真实网络搜索
2. 获取80-120条最新资讯
3. 智能过滤和分析
4. 生成结构化报告
5. 展示关键发现

## ⚙️ 配置说明

### 基本配置

编辑 `config_template.json`：

```json
{
  "company_info": {
    "name": "您的公司名称",
    "brand": "品牌名称",
    "industry": "所属行业"
  },
  "competitors": {
    "major": ["竞品1", "竞品2", "竞品3"],
    "emerging": ["新兴品牌1", "新兴品牌2"],
    "potential": ["潜在入场者1", "潜在入场者2"]
  }
}
```

### 自定义关键词

```json
{
  "keywords": [
    {
      "name": "本公司动态",
      "search_terms": ["公司名 最新", "品牌名 今日"],
      "priority": "critical"
    }
  ]
}
```

### 预警规则

```json
{
  "alerts": {
    "company_mention": true,
    "competitor_events": true,
    "negative_news": true,
    "crossover_entry": true,
    "tech_breakthrough": true
  }
}
```

## ❓ 常见问题

### Q: 如何修改监控对象？
A: 编辑配置文件中的 `competitors` 部分，或对AI说"增加竞品：XX公司"

### Q: 如何调整监控频率？
A: 修改配置文件中的 `schedule` 部分

### Q: 报告保存在哪里？
A: 默认保存在 `./reports/` 目录

### Q: 如何导出PDF？
A: 对AI说"将报告导出为PDF"

### Q: 支持英文吗？
A: 支持，修改配置中的 `language` 为 `en-US`
